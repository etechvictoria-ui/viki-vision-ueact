What is VIKI VISION?

AI-powered emergency dispatch decision support system. Helps dispatchers make critical decisions in emergency situations.

Key Features

⚡ Fast Decision Support

AI analyzes emergency data and provides instant recommendations.

🧠 JFP v5.2 Integration

Built on JFP protocol for deterministic AI behavior.

🖥️ Electron + React

Native desktop app with modern UI.

📋 Audit Logging

Complete audit trail of all decisions.

Quick Start

# Install dependencies
npm install

# Run development
npm run dev

# Build
npm run build
